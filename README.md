## Introducción
En el repositorio se encuentran laboratorios practicos 
- Fundamentos de NumPy y Pandas
- Visualización de datos con Matplolib y Seaborn
- Machine learning básico
- Introducción a deep learning

# Estructrua de la carpeta Notebooks 

### 01_Fundamentos_NumPy_Pandas.ipynb 
Ejemplos de manipulación de arrays con NumPy, Daaframe con Pandas.
### 02_Visualizacion_Datos.ipynb
Gráficos básicos con Matplotlib, Seaborn y Plotly.
### 03_Machine_Learning_Basico.ipynb 
Modelos de clasificación, regresión usando Scikit-learn.
### 04_Deep_Learning_Intro.ipynb
Modelo de red neuronal con TensorFlow/Keras, visualización del proceso de entrenamiento y evaluación del modelo.
